# XO_game
## create python venv 
 - Windows user:
  - py -m venv venv
 - linux user
  - python3 -m venv venv
## make sure that venv is activate
 - windows user:
  - cd venv 
  - Scripts\activate
 - linux user:
  - cd venv
  - source bin/activate
## pip install -r requierments.txt
to run this project :
type in terminal
py main.py
